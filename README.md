## AD_Lab_2330044_Experiment-5
Design and Implementation of a Natural Language Interface for Databases using LLM and MCP  – Application Development Lab
### Project Description
To design and implement a system that enables users to interact with a database using natural language queries by integrating a Large Language Model with Model Context Protocol (MCP), a Flask backend, and a web-based frontend.  
- HTML5
- Flask
- MySQL Workbench 8.0
- Visual Studio Code
- GitHub
### Files Included
- index.html - Webpage structure and layout
- app.py – Backend code and routing
- db_setup.py - Create the database schema and populate the MySQL database with sample data
- llm_to_sql.py - Convert user-provided natural language queries into corresponding SQL commands
